# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial release of Curl Keeper
- Save curl commands with name, service, and description
- Real-time search functionality
- Copy commands to clipboard
- Edit and delete commands
- Beautiful TUI interface
- Persistent JSON storage
- Multi-line curl command display with word wrapping
- Keyboard shortcuts for quick navigation

### Features
- `/` - Search commands in real-time
- `a` - Add new curl command
- `e` - Edit selected command
- `c` - Copy command to clipboard
- `d` - Delete command
- `Enter` - View command details
- `Ctrl+C` - Copy in search view (macOS: Cmd+C)

## [1.0.0] - 2025-01-XX

### Added
- First stable release
- Complete TUI implementation
- Cross-platform support (macOS, Linux)
- Comprehensive documentation
- Example commands
- CI/CD pipeline with GitHub Actions
- Automated releases with GoReleaser
